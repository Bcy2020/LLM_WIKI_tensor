## 放置规则

- 尽量有一个稳定的 canonical home。
- 跨领域页面可以带多个 `domains`，但最好仍有一个规范主路径。
- 页面要放在“以后最可能再去找它”的地方。
- 一般判断：
  - 定理 / 方法 / 数学对象 / 物理概念 → `concept`
  - 人 / 机构 / 仓库 / 数据集 / 软件 → `entity`
  - 读过的资料 → `source`
  - 值得沉淀的问题回答 → `question`
- `index`、`indices` 等数学/物理高频词默认不作为系统导航页链接依据。
